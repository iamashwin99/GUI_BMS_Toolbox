This archive includes files necessary to simulate parallel-connected-module and 
series-connected-module battery packs. It contains several utility functions from
the ESCtoolbox as well as the main simulation codes.  In particular,

- simPCM.m: Runs a PCM simulation and plots some sample results
- simSCM.m: Runs an SCM simulation and plots some sample results
For a description of the ESCtoolbox files, see comments in ESCtoolbox.zip.
